Request endpoints should clearly communicate data structure

https://nbtl.mesacc.edu/superuser/honors-api/stories/1

  • Collection (Parent of resource)
  • Resource (Child of collection)

  • Use keywords that well represent the data objects
    • schema
    • relationships
  • Be organized hierarchically
  • Should not be named after verbs
    • operations
    • actions
    • instead use methods
  • Be hyphen-separated (no underscores)
  • Should not require trailing forward slash
