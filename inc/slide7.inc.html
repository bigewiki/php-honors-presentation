"Scrum PM"

My API was designed to interact with a prebuilt database. This database was built to support the backend of a project management program. It loosely immitates JIRA and is influenced by agile development practices, specifically using the scrum model.

Agile Development
A methodology in software development for project management. Agile is characterized by:
  • Emphasizing user needs
  • Maintaining tight feedback loops
  • Quickly responding to changing requirements/priorities
  • Shorter release cycles and smaller updates
  • Deliver value quickly: add quality and expansion of features through iteratation
Scrum
An agile framework that plans work around cycles called sprints
  • Commonly 2 weeks long (deadlines)
  • Requires regular planning meetings, reviews, and standups (called scrums)
  • Clearer guidelines on planning, tracking, and prioritizing work
  • More focus on communicating problems, wins, and losses within the team
Work is meant to be organized into sprints, represented by stories, and broken up into tasks.
