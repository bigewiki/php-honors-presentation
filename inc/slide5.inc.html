HTTP Request Methods

Defines the type of the HTTP request, commonly found in the form of GET and POST
APIs should use methods to understand what operation to perform on the identified resource/collection

CRUD

Operation

Method
Create POST
Read GET
Update PATCH/PUT
Delete DELETE

PATCH VS PUT

PUT
Will replace an existing resource or create it if it isn't present. Expects to receive a complete object.
PATCH
Will only update specified properties and can only operate on existing resources.
