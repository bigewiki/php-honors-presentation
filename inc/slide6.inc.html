HTTP Status Codes

A status code is sent back in the response indicating whether the request was successful or produced an error.

100 Level Informational
200 Level Success
300 Level Redirection
400 Level Client Error
500 Level Server Error

Specific Status Codes

Code Meaning CRUD
200 OK Read, Update, Delete
201 Created Create
400 Bad Request All
403 Forbidden All
404 Not Found All
405 Method Not Allowed All
500 Internal Server Error All
