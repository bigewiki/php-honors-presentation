/stories/${NUMBER} (DELETE Request)

Use the story id in the path to delete that story. Will fail if story has associated comments, tasks, or child dependencies (foreign key check failure). Also requires token to be sent in header (key of 'token'). ex: /stories/34

RESPONSE

  • success: int
  • notice: string
